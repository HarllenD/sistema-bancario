# Sistema Bancário em Portugol

Projeto feito para a aula de lógica de programação usando o Portugol Studio.

## O que o programa faz

É um sistema bancário simples que roda no terminal. Dá pra criar contas, depositar, sacar, transferir dinheiro entre contas e listar todas as contas cadastradas.

## Como rodar

1. Baixe e instale o [Portugol Studio](http://lite.acad.univali.br/portugol/)
2. Abra o arquivo `Sistema_Bancario.por`
3. Aperte F6 para executar

## Funcionalidades

- Criar conta
- Depositar
- Sacar
- Transferir entre contas
- Listar contas e saldos

## Tecnologia

- Portugol Studio
